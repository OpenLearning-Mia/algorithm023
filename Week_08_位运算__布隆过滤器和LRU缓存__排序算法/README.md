#### 1.2. LRU缓存 的优点、应用场景

> 关键词：为啥要用它？它有啥好处？它能解决什么问题？？

```
1.优点：缓存淘汰策略，保留常用的，删除不常用的，高效利用缓存空间。

2.应用场景: 缓存

3.典型应用题：实现LRU缓存机制

4.LRU缓存定义：是一种缓存淘汰策略；计算机的缓存容量有限，如果缓存满了就要删除一些内容，给新内容腾位置。但问题是，删除哪些内容呢？一般希望删掉那些没什么用的缓存，而把有用的数据继续留在缓存里，方便之后继续使用。LRU缓存就是来解决这个问题。
```
